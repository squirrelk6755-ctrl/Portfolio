## ✨ Features

*   **Light Mode Design:** A bright, cinematic interface with a soft, drifting pastel ambient background.
*   **Interactive 3D Project Cards:** Smooth flip animations that seamlessly reveal technical features and metrics upon hovering.
*   **Scroll Reveal Animations:** Content gracefully fades and slides into view as the user scrolls down the page.
*   **Glassmorphism UI:** Subtle translucent elements with soft shadows and borders for a modern, refined look.
*   **Fully Responsive:** Optimized for an excellent user experience across mobile, tablet, and desktop screens.
*   **Custom Ambient Animations:** Includes slow-moving background elements like drifting clouds and a high-altitude airplane to create visual depth without distraction.

---

## 🛠️ Tech Stack

*   **HTML5:** Semantic and accessible structure.
*   **Tailwind CSS:** Utility-first styling (implemented via CDN) for rapid, responsive, and highly customized UI design.
*   **Vanilla JavaScript:** Lightweight DOM manipulation for intersection observers and subtle scroll effects.
